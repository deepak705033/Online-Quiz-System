# Online Quiz System

## Description
This is a python based online quiz system that uses a CSV files to store the questions and record of each students

## Features
Enter Name and Class of Each student
Only greater than or equal to 10th class students is participate in quiz
Automatic calculate score
Store Student record and question in CSV file

## Technology used
Python
Excel or CSV file
File handling

## How to run
1. Install Python
2. Download the project
3. Run the code
